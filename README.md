# Setup

1. Setup the global environment:
    ```
    sudo npm --global node-dev bower
    ```

1. Setup local node environment:
    ```
    npm up
    ```

1. Setup local bower environment:
    ```
    bower up
    ```

1. Start the app:
    ```
    npm start
    ```

1. Open the site on your browser:
    ```
    http://localhost:3000/
    ```
